Apuntes

      tipos de posicionamiento : 
      * static ( por defecto)
      *relative
      *absolute
      *fixed
      *sticky
      
      para mover los elementos posicionados se activan las propiedades: 
      * top ( vertical - eje y )
      *bottom ( vertical - eje y)
      *left ( horizontal - eje x)
      *right ( horizontal - eje x)
      *z-index ( profundidad- eje z)
      
      Las propiedades top y left tiene prioridad sobre bottom y right respectivamente
      
      ninguna de estas propiedades funciona si el valor de position es "static"

Posicion en z

Volver al indice 
        Es la propiedad que premite controlar la profundidad de los elementos posicionados, su valor por defecto es auto, acepta numeros positivos, negatvos y cero.

        A mayor valor el elemento esta mas al frente, a menor valor mas al fondo.

        Un elemento padre nunca podra estar sobre sus elementos hijos, sin embargo os elementos hijos si pueden dandoles un valor negativo.

        Es recomendable trabajar con valores los cuales te permitan trabajar con numeros intermedios ( 10/100 - 100/100)
