Flex-grow y Flex-Shink

      Notas:

      Flex-grow se usa para determinar el factor de crecimiento de los elementos segun el espacio del contenedor padre.

      flex-grow: 0 | 1 | 2

Flex-grow y Flex-Shink

      Notas:

      Flex-shrink se usa para determinar el factor para encoger los elementos segun el espacio del contenedor padre.

      flex-shrink: 0 | 1 | 2
  
      Para eliminar la caracteristica de encogerse de los elementos el valor pre-determinado es  0

     flex-shrink= 0;

Flex-basic

        Notas:

        Determina el tamaÃ±o del hijo dentro del main axis del padre ( columna o fila) 

        Si el contenedor tiene propiedad  Flex-direction: row  flex-basis representara el width de los elementos hijos.

        Si el contenedor tiene propiedad   Flex-direction: Column flex-basis representara el height de los elementos hijos.

        el Shorthand para estas 3 propiedades seria  flex : grow(0)|shrink (1)|basis (auto)
