Unidades de medida

Notas :

  • Existen 2 tipos de unidades generales , absolutas y unidades relativas; ejm de unidades absolutas: pc, mm, in, pt(1/72 in), px ( 1/96 in)
  • El tamaño de la fuente preeterminada de los navegadores de 16px
  • Las unidades relativas son aquellas cuyo valor dependera de su elemento contenedor o contexto . ejm: em, rem, ex, ch, %, tamaño de viewport ( vw, vh, vmin, vmax):
  • Para mas info visitar www.pxtoem.com
  • Es recomendable que para tamaños de fuentes se usen unidades relativas.
ooo

Pixels

La unidad de medida pixels se considera una unidad de medicion absoluta, es una medida que tiene un valor en el mundo real que equivale a 1/96 pulgadas y dicha medida no es variable

Variables y Custom Properties en Css

Notas :

  • Css al igual que los lenguajes de programacion, tiene un apartado de variables, las cuales tienen comportamientos similares, como lo es el alamacenamiento de informacion general
  • El scope de accion de estas variables se limita al contenedor donde fue declarada ejm: 
                
   #container{
     --primary-color="#fff"
   }
   .item-container{
     background-color=var(primary-color)
   }
  • Para declarar una variable en Css se usa el doble guion medio "--nombre de variable"
  • Para llamar a una variable en Css se usa la palabra reservada var( --primary-color, Parametro 2 (opcional) ) , la cual acepta 2 parametros separados por una coma, siendo el primero el nombre de la variable en cuestion y el segundo un respaldo el cual se ejecutara en caso de que la variable no contenga ninguna informacion o no se pueda procesar
  • Para declarar variables globales se puede hacer uso de la pseudo clase :root la cual se aplica al documento general HTML, esta es la opcion mas usada y recomendada para aplicar estilos generales y que su alamacenamiento no afecte al documento en cuestion.
  • Existen distintos tipos de funciones dentro de CSS, ejm:
    • variables = var ()
    • Links o url para llamar documentos externos = url()
    • Calculos aritmeticos = calc()
    • Valores minimos y maximos = min() & max()
      • Esta funcion sirve para establecer comparativas entre los valores asignados a la variable, tomando como referencia el mayor o el menor dependiento de la funcion que se use
      • El uso de estas funciones se recomienda para realziar cambios dinamicos sin uso de @mediaquerys ya que con esta funcion una propiedad podria tener distintos valores asignados y por lo general se debe realizar la comparativa entre valores absolutos y relativos.
      • En el caso de min(20px, 25vh), esto quiere decir que tomara como valor el minimo entre los 2 parametros asignados. Cuando el valor de 25vh sea mayor a 20px, el valor asignado sera 20 px.
    • Valores minimos, medios, maximos = clamp( min | med | max )

Propiedades de texto y fuentes

Notas :

  • Son todas aquellas propiedades que afectan a las tipografias:
    • font-family = Nombre de la familia tipografica
    • font-size = tamaño de la tipografia
    • font-style = tipo de estilo ( cursiva,oblicua, normal)
    • font-variant = variacion entre mayusculas y minusculas
    • font-weight = peso de la tipografia
    • line-height = altura de linea o interlineado
  • Para importar archivos tipograficos personalizados o externos se ua la regla en ccs de @font-face{} .
    1. El primer parametro que se tiene que aplicar sera el nombre de la familia tipografia ejm : font-family : "nombre de la tipografia a agregar"

      " Es recomendable el uso de una segunda familia tipografica precargada en caso de que la tipografia que se quiere importar no pueda cargar "
    2. Para poder cargar una tipografia externa al documento, se usa la propiedad src = url ("../nombre de la carpeta / nombre del archivo") ( el archivo tiene que estar guardado en formato woff | woff2 (Web Open Font Format) )

Propiedades de bordes y outlines

  • Para lograr un elemento circular mediante los bordes se puede usar la propedad = border-radius: 50%;, esto solo aplica para elemtos de caja "perfectos"
  • Para crear una elipse las medidas de la etiqueta Border se separan mediante un signo de division " / " border-radius: 1 rem / 3 rem
  • La propiedad outline se aplica como un borde exterior al elemento, separado de lo que son los elementos Box model (elementos de caja).
  • La propiedad Outline empuja los elementos alrededor de la caja hacia afuera, pero no el contenido del Box Model.
  • La propiedad Outline-offset se aplica para poder separar el elemento del Outline.

Propiedades de background

Notas:

  • background-color: define el color de fondo del elemento
  • background-image: define la imagen de fondo del elemento
  • background-repeat: define cómo se repite la imagen de fondo en el elemento.
  • background-size: define el tamaño de la imagen de fondo, primer valor Ancho, segundo Alto.
    - cover: cambia el tamaño de la imagen de fondo para asegurarse de que permanezca completamente visible.
    - contain: cambia el tamaño de la imagen de fondo para asegurarse de que el elemento esté completamente cubierto.
  • background-position: define la posición de la imagen de fondo, primer valor x, segundo y, si no se especifica un segundo valor éste será center aparte de valores numéricos podemos indicar el posicionamiento con las palabras: center, top, bottom, left and right
  • background-clip: define cuánto debe extenderse el fondo dentro del elemento.
    ---- border-box El fondo se extiende hasta el borde exterior de el contenedor .
    ---- padding-box El fondo no se muestra a través del borde (el fondo se extiende hasta el borde exterior del padding).
    ---- content-box El fondo se dibuja desde donde inicia el contenido.
  • bg-origin: define el origen de la imagen de fondo.

  • Para añadir multiples fondos ( imagenes ) se puede hacer a travez de la etiqueta background : (primer fondo) , (segundo fondo), ...

Propiedades y estilos de imagenes

Notas :

  • Para hacer que una img sea responsiva, como recomendacion en los resets del documento CCS es aconsejable colocar img { max-width : 100% ; height : auto; } .
  • Los elementos IMG se comportan Inline
  • Existen 2 prpiedades para ajustar el tamaño de una imagen en un contenedor con tamaño definido:
    object-fit ( none | contain | cover | fill (valor por defecto )
    object-position : (ubicacion con palabras clave o medidas absolutas ) | sacle-down ( escala la imagen comparando los valores de none y contain, tomando el menor entre ambos )
