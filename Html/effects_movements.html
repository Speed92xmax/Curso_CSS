Sombras


  • La propiedad Box Shadow se compone : box-shadow : valor-X | valor-Y | blur-radius | spread-blur | color | inset/outset
  • Si no es especifica el color en la propiedad, este tomara como valor hereddo el color de texto del elemento.
  • Acepta valores negativos
  • Las sombra por defecto son outset sin embargo este valor no se debe de escribir si ya esta estblecido ya que tomara como que esta sombra no es valdia.
  • La propiedad text-shadow acepta como parametros : val-X | val-Y | blur | color.
  • Para agregar sombras multiples solo hay q separar los valores de cada sombra con una ",".

Degradados


  • La propiedad linear-gradient: se compone de : grados | color y todos los aprametros van separados por una ",".
  • El punto " 0 " de los grados esta ubicado en el centro / inferior del objeto al que se le aplica.
  • La unidad en la que se establcen los grados es "deg".
  • Se pueden usar palabras clave to right| left | top | bottom y se puede hacer combinaciones entre ellas : to bottom left
  • Se puede usar los procentajes para ubicar la proporcion que podria abarcar el color ejmp : red 0% 15% , green 16% 80%, white 81% 100%

Filtros de imagenes


  • Para aplicar las propiedades de effectos, se utiliza la palabra reservada filter.
  • El filtro de Brightness (brillo) va dese 0 ( sin luminocidad ) a 1 ( 100% iluminado ). Un valor superior a (1) crearia efecto de sobre exposicion.
filter : blur (0.1 rem)
filter: brightness(0.5)
filter: contrast(0.3);
filter: grayscale(0.8);
filter: sepia(0.7);
filter: hue-rotate(180deg);
filter: invert(1);
filter: opacity(0.3);
filter: saturate(5);

Filtros multiples de imagenes


  • Para aplicar filtos multiples a una misma imagen, se tienen que separar dichos filtro ( o funciones) por un espacio en blanco " ".
filter: blur(0.1rem) invert(1) saturate(4) hue-rotate(180deg);

Estilos de mezcla


  • La porpiedad para realizar las mezclas entre 2 imagenes superpuestas es mix-blend-mode:
  • Dicha propiedad se tiene q aplcar a las 2 imagenes q se van a ver afectadas ( o a todas las img dentro de contenedor padre )
  • Cada imagen va a reaccionar de forma distinta ya que el algoridmo que realiza estos calculos lo hace sobra cada imagen por separado

mix-blend-mode: multiply;
mix-blend-mode: color-burn;

Mascaras de recorte


  • Para realizar una mascara de recorte circular se usa la propiedad clip-path: circle (100px | 100px at top | 100px at top right | 100px at 60% 20%)
  • Para mascaras en forma rectangular se usa la propiedad clip-path: inset (hay que establecer un valor incial)
  • Para mascaras de forma irregular ( por nodos ) se usa la propiedad clip-path: polygon(0 0, 100% 0, 50% 100%);
  • Pagina de referencias para poligonos irregulares : bennettfeely.com
clip-path: circle();
clip-path: inset(3rem round 1rem);
clip-path: polygon(custom);

Scroll Snap


  • Para realizar este efecto, obligatoriamente el elemento contenedor tiene que tener una altura definida.
  • El elemento contenedor tiene que tener activa la propiedad overflow: scroll .
  • La propiedad tiene como nombre scroll-snap-type: CONTROL [x (inline) | y (block) | both] - COMPORTAMIENTO [mandatory | proximity] y esta se aplica al contenedor padre.
  • Todos los contenedores hijos tienes que tener activa la propiedad scroll-snap-align : start | center | end ; la cual define el punto de referencia en el cual se va a realizar el scroll.

Transiciones en css


  • La propiedad trantision es el shorthand para:

    • transition-property ( propiedad a la que se le aplicaria transicion )
    • transition-duration ( duracion de la transicion )
    • transition-timing-funtion ( velocidad : ease | ease-in , etc )
    • transition-delay (retraso previo al incio de la transicion )
  • Para plicar transiciones a distintas propiedades se puede hacer de 2 modos :

    • Separar cada propiedad por una " , " coma, en la propiedad transition-property // transition-property: background-color, border, border-radius;
    • Utilizando la palabra reservada "all" en el shorthand.

Hoover me

transition: background-color 350ms ease-in-out 100ms;
transition: all 550ms ease-in-out 50ms;

Transformaciones 2D


  • Las transformaciones en CSS se dividen en Transformaciones 2D y Transformaciones 3D y se llaman con la propiedad transform
  • Acepta cualquier valor de unidad, y valores negativos
  • Las transformaciones de movimiento 2D son transform : translate(x | y) | scale(x | y) | rotate(x | y) | skew(x | y) | matrix (
transform: translate(-5%, -5%);
transform: scale(1.2);
transform: scaleX(-1); ( efecto flip )
transform: rotateZ(180deg);
transform: skewX(-20deg);
transform: scale(1.2) scaleX(-1);

Transformaciones 3D


  • Para hacer uso de las transformaciones en 3d se tiene q activar la propiedad Perspective la cual se puede aplicar al contenedor padre, o declarar dicha funcion como prefijo antes de realizar la transformacion ejm: transform: perspective(valor de profundidad de campo) translateZ(20%);
transform: translate3d(-50px, -50px, -500px);
transform: scale3d(2.5);
transform: rotate3d(1, -4, 0, 30deg);
transform: rotate3d(0, -8, 0, 60deg)
transform-origin: bottom left;

Animaciones


  • Para realizar una animacion se utiliza la propiedad animation cuyos parametros pueden ser:

    • Name : Nombre asignado de la animacion
    • duration : Duracion.
    • timing-function : curva bezier de movimiento.
    • delay : Retraso.
    • iteration-count : Numero de repeticione.
    • direction : Orden de ejecucion de la animacion.
    • fill-mode : Decide si conserva los cambios de propiedades al final de la animacion.
    • play-state : Estado de la animacion pentre pausado y running.

  • Los fotogramas se invocan con la propiedad @keyframes (nombre de la animacion)
  • La Propiedad @keyframe obligatoriamente tiene que tener 2 fotogramas incluidos from () {fotograma inicial} || to () {fotograma final}
  • Para incluir multiples fotogramas, los @keyframe se entienden como un valor de porcentaje (%) dentro de una linia de tiempo , ejm: 0% { filter: 1} 50% | { filter : 2} | 100% {Filter : 3}
